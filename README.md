# 符籙 · Fulu

道家符籙之標準庫——符式・出典・白描（TypeScript canvas，文言 API）。
承粉本庫（fenben）之法統：格網為律、出典為戒、筆觸為器。

**研究陳列，非行持之符。** 本庫為圖像學與書法研究之藏；符籙之「用」屬道門
儀軌，不在此庫。

- `src/fuge.ts` 符格（紙段之律）｜ `src/fupu.ts` 符譜（式與出典）
- `src/bi.ts` 筆具（承粉本：三等筆・一筆提按・蔽法）｜ `src/fubi.ts` 符筆（紙框・硃層・塊鑿）
- `src/fu/` 逐符親筆｜ `tools/幀符.cjs` 逐符截圖（`node tools/幀符.cjs [id]` → 圖錄/符/）
- `docs/考據/` 考據卷｜ `參攷-local/`（gitignored）本地參攷圖｜ `index.html` 陳列頁（先 build）
