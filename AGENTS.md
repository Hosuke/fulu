# AGENTS.md — 符籙庫匠人紀律（Codex 所讀）

此倉為道家符籙標準庫（TypeScript canvas 白描）。汝之職分與戒律：

## 汝可為
- 依 goal 修碼、審碼、跑 build 與 headless 驗證；
- 雲篆結構之數值審（筆畫交接距・封口・對稱——goal 給數值即算之）；
- 於 `匠筆-codex/` 內自運競本（goal 明命時）。

## 汝不可為
- **不得改 `src/fu/` 逐符之筆與 `src/fubi.ts`**（符身親筆之律——主坊落筆，
  除非 goal 明言「審稿直改」）；
- **不得改 `fuge.ts` 符格數值**（繫出典，一字不得動；疑錯報之勿改）；
- **不得杜撰符式**：網上「靈符」多今人偽作，凡引必道藏本／善本圖版；
  查不實寧空勿補；
- 不得引入外部依賴・外部資源請求；不得 commit（主坊統籌）。

## 碼之律
- 漢字標識符有意為之，勿「修正」；
- 白描無填、destination-out 蔽法、三等筆（骨0.34/衣0.21/細0.14×W_OUT）；
- tsc strict 過乃交；headless 截圖親見乃報（勿憑座標推算「應該對」）；
- 截圖法：畫於透明布，末合成深底（在畫布內先鋪底則蔽法鑿穿露白）。

## 驗收語彙
「錨中」＝符格段位合 fuge 數值；「筆連」＝雲篆筆畫交接≤容差；
「式合」＝與 fupu 條目出典之式相符；「氣韻」＝主坊目審之域，勿自評。
